CNC File Management System - User Guide

Introduction

Welcome to the CNC File Management System. This platform allows users to efficiently manage files related to CNC drawings, track revisions, and perform other essential operations such as uploading, downloading, and deleting files.

This guide will walk you through the various features and functionalities available in the system.

System Features

  • File Browser: View and manage all files in the system, including details such as filenames, descriptions, and creation dates.
  • File Uploading: Users with appropriate permissions (Admin, Engineer, Programmer) can upload new files to the system.
  • File Management: Delete files (restricted to authorized users).
  • Search and Filter: Search and filter files based on various criteria, such as filename, drawing number, and description.
  • Recent Files: Easily access recently uploaded files through the dashboard.

How to Use the CNC File Management System

Logging In

  1. Navigate to the login page.
  2. Enter your username and password.
  3. Click Login to access the system.
  4. After successful login, you’ll be directed to your Dashboard.

Dashboard Overview

  • Welcome Message: Displays a personalized greeting with your username.
  • Navigation Menu: Use the sidebar to navigate between the Dashboard, File Browser, and File Upload sections.
  • Recent Files: View the latest files you have uploaded or worked with.

File Browser - Manage Files

The File Browser allows you to view and manage files within the system.

Accessing the File Browser

  1. In the Sidebar, click File Browser to access the file management page.
  2. You’ll see a list of files grouped by their drawing numbers.

Search and Filter

You can search for files by filename, drawing number, or description:

  • In the Search Bar, type keywords to find specific files.
  • Use the Filter button to narrow down files by date, uploader, or drawing number.

Downloading Files

  1. In the Actions column, click the Download button next to the desired file.
  2. The file will be downloaded to your local machine.

Deleting a File

  1. In the Actions column, click the Delete button next to the file.
  2. A confirmation dialog will appear to confirm that you want to delete the file.
  3. Click Yes to permanently delete the file.

Note: Deleting files is restricted to Admins, Engineers, and Programmers.

Uploading Files

The Upload File section allows users with the appropriate role (Admin, Engineer, Programmer) to upload new files.

Steps to Upload Files

  1. In the Sidebar, click Upload File.
  2. Fill in the required fields:
    • Select File: Choose the file you want to upload.
    • Description: Provide a brief description of the file.
    • Revision Number: Enter the revision number for the file.
  3. Click Submit to upload the file.
  4. You’ll receive a confirmation message if the upload is successful.

Note: File uploads are restricted to Admins, Engineers, and Programmers.

User Roles and Permissions

  • Admin: Full access to all files, including uploading, editing, deleting, and managing users. Can manage file metadata and revisions.
  • Engineer: Access to view, edit, and upload files. Can delete files and upload new versions of files.
  • Programmer: Similar to Engineer, with the ability to upload, edit, and manage files. Can delete files.
  • User: Can only view and download files. Cannot upload, edit, or delete files.

Contact Support

If you encounter any issues or need further assistance, please contact support@cncfilemanagement.com.

Conclusion

The CNC File Management System provides an easy-to-use interface for managing files, tracking revisions, and facilitating collaboration among users with different roles. Whether you're uploading files or simply browsing, this system streamlines your CNC file management process.
